# CatchTheBlue


# Game Overview

## I created a game with simple rules:

- ✅ **Click the blue circle** to earn **+1 point**
- ❌ **Click anything else** and you’ll get **-1 point**
- 🔼 Every **5 points**:
  - You advance to a **new level**
  - Circles move **faster**
  - New **penalties** are introduced to make things more challenging
